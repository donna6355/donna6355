### 플러터 개발자 전다나 😼

과거에는 국제 행사 기획자로서 일했다가, 프론트엔드 웹개발로 개발을 시작해서,<br />
현재는 플러터와 다트로 크로스 플랫폼을 탐험하는 4년차 **우당탕탕 개발자**<br />
정확한 문제 파악부터 해결 방안까지 고민하는 **프로 문제 해결러**<br />
사고뭉치 반려묘에게 잡혀사는 **호구 집사**<br />
